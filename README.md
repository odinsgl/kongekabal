# King's Solataire

How to play:

* Click new game
* Click the card pile to draw one card to each stack.
* If you there is a card on top of one of the stacks that is the same suit and higher value than the card on top of another stack, you can click the lower value card to remove it.
* If there is an empty slot, and one or more stacks has multiple cards, you have to move one of the cards to the empty slot before drawing again.
* You win when the draw pile is empty and there is one king in each stack.

Good luck!
